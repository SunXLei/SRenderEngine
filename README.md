# SRenderEngine

## Introduction

![test](./image/test.png)

SRenderEngine is a render engine written in opengl and c++11. This project is basically for me to learn opengl and graphics technics. It is still in a very early stage.



## Main Feature

- Forward Rendering
- Deferred Rendering
- PBR metallic-roughness workflow
- Shadow mapping
- PCF
- Tonemapping
- SSR
- ...
